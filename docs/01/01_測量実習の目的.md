# 1.1 測量実習の目的

測量とは我々の住んでいる世界の中にある様々な物の形態を、ある目的のために測定する技術である。土木建築施設の計画・設計・施工・管理、都市・地域計画、都市・地域環境管理、地球環境管理、などに必要不可欠な情報を提供するものであり、土木・環境の分野においてきわめて重要な技術である。

測量を習得するためには、講義による理解だけでは不十分であり、自らの体躯を使い測量を実習することによってはじめてその習得が期待できる。測量実習の目的は単に測量機器の操作の習熟に止まらず、測量を効率良く行うために必要な作業計画や段取りのたて方の習得、測量学の知識についての深い理解にあり、さらに最終目標としては、測量結果に対して適切な判断が下せるような総合的な能力を養うことにある。

測量にはその目的や方法の相違によって種々のものがあるが、本実習では主に基本となる測量を習得することを狙いとしている。具体的には、はじめに基本となる測量手法として角測量、距離測量、水準測量、および、コンピュータを用いた座標計算と地形図作成の基礎、を学ぶ。ここでは主に、トータルステーションとティルティングレベル、GIS（Geographical Information System）によるコンピュータを用いた地形図作成の操作方法を実習する。その後、これらの手法を駆使して縮尺１／２００程度の地形図作成を目的とした地形測量を実習する。

本実習は「測量学の講義内容の十分な理解」と「本実習指導書の予習」を前提として授業を進めていく。本実習における測量作業基準の取り扱いは、国土交通省等の政府の省庁、地方公共団体が行う測量、いわゆる公共測量と称されるものに基づく規準に従うことを原則とする。
