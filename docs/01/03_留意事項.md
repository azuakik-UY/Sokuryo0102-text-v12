# 1.3 留意事項

- 体調のすぐれない場合（特に熱が37.5度以上）の場合は速やかに大槻（kotsuki@yamanashi.ac.jp）まで連絡すること。

- 全体への連絡はCNSコミュニティを用いて行う。

- 計算を行う実習では各々のPCを持参するBYOD（Bring Your Own Device）を基本とする。

- 双方向ライブ講義やMoodleでの動画視聴を用いる場合がある．

- 本科目は、JABEEプログラムの学習・教育目標達成に対して以下のような位置づけである。＜測量学＞→＜測量学実習第一，第二＞

- 測量学実習第一は、測量学実習第二（木曜日IV時限）と同時に履修申告しなければならない。授業は木曜日のIII時限～IV時限の連続で行う。

- 雨天の場合は講義を順延とする。また、所定の精度を満たさなかった場合は再度測量をやり直す。そのため夏休みに入って約１週間は授業を行う可能性があるので、終日予定を入れないでおくこと。

- 予習の準備不足等で授業時間内（IV時限まで）に作業が終わらない場合は、作業が終わるまで実習を行う。
