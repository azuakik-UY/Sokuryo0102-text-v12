# 1.6.5 Excel のインストール

- 実習ではオンライン版Officeを使用しないこと。

- 山梨大学総合情報戦略機構HP「Microsoft 365包括ライセンス（A3）」：  
  <https://sojo.yamanashi.ac.jp/services/software/microsoft-a3/>  
  の「個人用ライセンス」を見ながら、Officeの最新版を自分のPCにインストールすること。

- Windows版

1.  スタートボタンを押し、サインインしているユーザーのアイコンをクリックして、「アカウント設定の変更」を選択。ユーザー名の下に「管理者」と表示されていることを確認する。それ以外の表示になっている場合には、管理者ユーザーでサインインし直す。

2.  「スタート」ボタンを右クリックし、「アプリと機能」をクリックする。  
    一覧中に「Microsoft Office」があった場合はアンインストールして、Windowsを再起動する。

3.  こちらのリンク（<https://aka.ms/o365id>）にアクセスし、「Office」の言語が「日本語（日本）」、バージョンが「64ビット」（あるいは32ビット）であることを確認し、「Officeのインストール」をクリックする。

4.  ダウンロードされた「Office Setup.exe」をクリックするとインストールが開始される。

5.  「すべて完了です。Officeはインストールされました。」の画面が表示されたら「閉じる」をクリックする。

6.  インストールされたOfficeアプリ（Word, Excel、 PowerPoint）のいずれかをクリックして起動すると、「サインインしてOfficeを設定する」の画面が表示されるので、「サインイン」ボタンをクリックする。

7.  表示されたアカウントの入力画面にて、山梨大学のアカウント（@yamanashi.ac.jp）でサインインする。

8.  サインイン完了後、「すべてのアプリにサインインしたままにする」という画面が表示された場合、「組織がデバイスを管理できるようにする」のチェックを外し、「いいえ、このアプリのみにサインインします」をクリックする。
