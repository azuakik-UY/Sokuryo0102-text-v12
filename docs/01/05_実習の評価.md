# 1.5 実習の評価

- 成績評価：

シラバスに基づき、受講態度とレポートを50:50で採点する。評点については、出席日数、レポート・図面の採点、実習の態度をもって行う。実習の態度とは、器具の取り扱いや管理・後片付け、作業のスムーズさ（班員との協力体制）、作業の的確さ等を総合的に考慮したものである。

- 出欠：

４／５以上の出席が必要である。したがって、おおよそ2.5回の欠席で不可となる．遅刻・早退は0.5回の欠席として取り扱う。ただしレポートを作成するためには実習を受ける必要があるため、事実上、欠席は単位認定不可に直結することに留意すること。

- レポート

  - Moodleからの電子データでの提出を基本とする。詳細はMoodle上の指示に従うこと。

  - 締切

    - レポート課題が設定された6日後（水曜日）の24時を締め切りとする。ただし、レポート（#4）は実習当日に提出とする。未提出レポートが１つでもある場合は単位不可となる。レポート・図面の提出は指定された期日を厳守すること。

  - 剽窃厳禁

    - 写したと判断されたレポートは未提出レポートとする。写させたレポートも未提出レポートと見なされ、単位不可となる。

    - ただしレポートフォーマットや自ら作成したレポートからの転記は作業の効率化であり、積極的に行うべきである。

  - 再提出

    - レポートは提出され次第直ちに簡易判定を行い、一定の水準に達していないものは再提出対象とする。
