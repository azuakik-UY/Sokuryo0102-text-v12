# 3.4.2 電源を入れる

- 電源キー\[ON\]を押す。電源が入ると、自己診断が行われ、この画面が表示されます。その後、画面は測定モードになる。

- 「チルトオーバーレンジ」が表示された場合は、機械が、傾斜角補正範囲を越えて傾いている。もう一度、整準を行うこと。その後、再度水平角、高度角を表示させること。
