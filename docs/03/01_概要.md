# 3.1 概要

測量を行うには、行う測量の目的に適した器具や測定方法を的確に選び、自由に器具を取り扱うことができなければならない。ここでは、本実習で用いる測量器具の取り扱い方法の概略の理解を目的として、角度および距離を測定するトータルステーション、簡易的に距離を測定する巻尺、高低差を測定するレベルに必要な測量器具について基本的な取り扱い方法を習得する。なお、ここで説明する測量器具と測量内容の関連を表 3.1に示す。
