# 2.2.4 選点

踏査によって得られた知識に基づいて測量実施計画を立て、以下のことに注意して測点などの測点を設ける位置を定める。

- 測量作業は大学構内および大学周辺で行うので、作業を行う上で大学の研究・教育の妨げや交通の障害とならない所に測点を設ける。

- 測点間の見通しのよいこと。

- 観測結果の処理製図・報告書の作成：内業

- 踏査選点造標観測：外業

- 計画準備

  - 測点が測量終了まで安全に保存されること。

  - 測点測量では、続いて行われる細部測量などに便利に利用できる点であること。

  - 測定する辺長に著しい長短がないように配慮し、測量結果の調整計算が合理的に行えるようにする。
