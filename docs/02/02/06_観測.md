# 2.2.6 観測

- 使用器具は、測定を開始する前に点検および調整を必ず行う。

- 観測者が無理のない姿勢で観測できる高さに機器を据え付け、致心および整準作業は確実に行う。

- 各自が作業できるように、作業分担を適宜交替する。

- 観測作業開始前に野帳に記入欄を作っておく。

- 観測値を野帳に記入する場合には、観測者の読みを必ず復唱して記入し、また野帳は誰が見てもわかるように一定の方式に従って記帳すること。訂正は、消しゴムを使わず斜線を引いて行う。

- 測定が終了したら、測定結果をその場でチェックし、誤りがあれば直ちに再測する。

- 作業は時間割の枠内で終了できないこともあり、また、雨天の際には作業ができないことを考慮し、計画的に余裕を持って作業を進めること。なお、時間割の枠内で実習が終了できない場合には、放課後や空き時間に作業を行うことになる。
