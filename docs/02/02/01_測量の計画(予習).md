# 2.2.1 測量の計画（予習）

- 実習を開始する前に各自または各班が考えなければならない点は以下のとおりである。

  - 測量の目的は何であるか。

  - どの程度の精度が必要となるか。

  - どのような測量機器を用いなければならないか。

  - 測定回数や測定方法はどのようにしたらよいか。

  - 各観測間の精度はどのように処理するか。

  - 誤差を生ずる主な原因には何があるか。

  - 誤差を補正し、誤差を調整するにはどうしたらよいか。

  - 測定値の点検は測量作業上のどこで行うか。

  - 時間はどのくらい必要か。

  - 野帳（観測値を記入するための用紙）の準備
