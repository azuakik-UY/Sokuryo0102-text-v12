# 2.4.2 図面の表現方法

## 作成の仕方

- Word上で図面（図形）を直接挿入すると不都合が生じやすい。PowerPoint等の別のソフトウェアで図面を作成し、それを「図として貼り付け」でWordに張り付ける方法を勧める。

## キャプション

- 図-\*・・・、表-\*・・・のようにキャプションを付けること。

- キャプションは、図の場合は図の下に、表の場合は表の上につけること。

- 適切な表題を何という図面か（例えば、「平面図」「横断図」）、だれが図面を見ても容易に理解できるように名付ける必要がある。

- 文字サイズは、本文と同じ10.5ptを基本とする。

## 単位の明記

- 図表中の物理量の単位を明記すること。

## 縮尺と北向き記号

- 測量成果には、縮尺と北向き記号を備えることが必要となる。縮尺は指導書で指定されるが、一般に縮尺は区切りの良い数とする。例えば、1/100、1/150、1/200などとし、1/135などとはしない。

- QGISでの配置については、8.5.8　地形図を完成させる で説明する。

## 線の太さ

- 線の太さに変化をつけ、分かり易い図面とすること。QGISでの線の太さの設定方法は、8.5.6　等高線で説明する。

## 文字の大きさと書体

- 図面上の文字は図面の大きさに適した大きさ（9pt以上が目安）とし、ゴシック体あるいは明朝体の書体で書き入れること。
