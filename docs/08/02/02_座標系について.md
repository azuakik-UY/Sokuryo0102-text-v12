# 8.2.2 座標系について

GISは位置情報を持ったデータを扱う。地理空間上の場所を取り扱う上では、どのような座標を用いているのかを明確にしなければならない。

表 8.2に示すように、データの持つ座標系と、表示するのに用いる投影座標系とがある。また、表 8.3のように、座標系には、地球上の位置を緯度／経度を用いて度分秒（あるいは度）で表すものと、地球上のある測点から南北・東西に平面座標を取ってメートル単位で表すものとがある。これらは非常に混同しやすいので注意が必要となる。

本実習では練習用データも含めてメートル単位で測量している。今回はこれらの基準点は公的な平面直角座標系とは異なるが、今回は仮に「JGD2011 / Japan Plane Rectangular CS VIII（EPSG:6676）」という平面直角座標系を用いているが、地球上の位置は気にせずに作業を行う。別途、参照点の測量を実施すれば、本測量を地球上の位置として取り扱うことができるようになる。参照点の測量は、RTK-GNSSと呼ばれる高精度の測位機器を用いて行なわれる。
