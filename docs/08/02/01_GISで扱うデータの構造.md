# 8.2.1 GISで扱うデータの構造

GISは位置情報を持った様々なデータ（地図など）を重ね合わせて様々な分析・表示を行う。GISはWordやExcelのファイルと異なり、<span class="mark">単独のファイルで機能するものではない</span>点を理解する必要がある。QGISで扱うデータの種類、拡張子とデータの内容について、構成表を表 8.1に示す。QGISを例にすると、拡張子.qgzのプロジェクトファイルが、地物を表すデータとの紐付けと表示方法を管理している。具体的な地物は、ベクタデータやラスタデータでそれぞれ単独に存在している。テキストデータはそれそのものではどのようなタイプの地物を表しているか不明であるため、QGIS上で定義しベクタデータかラスタデータとして読み込めるようにしなければならない。8.5.4　で行っているのはこの操作である。
