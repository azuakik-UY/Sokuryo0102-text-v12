# 8.5.2 QGISの起動とファイルの保存

- - 
  - 
  - 

QGISの起動画面左下の「スタート」検索窓に「qgis」。アプリ一覧から「QGIS\*.\*\*」「QGIS Desktop\*.\*\*.\*」をクリックして起動する。<img src="../../assets/media/media/image97.png" style="width:1.26312in;height:1.9685in" />

- - 
  - 
  - 

プロジェクトの保存画面上部の「プロジェクト」「名前を付けて保存」⇒作成した「qgis」フォルダを選択ファイル名を「t\*\*ce\*\*\*\*.qgz」（\*の部分は学籍番号）として保存する。<img src="../../assets/media/media/image98.png" style="width:5.31597in;height:2.12986in" />
